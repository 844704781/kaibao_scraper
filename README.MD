# 🎉 开宝体育赛果数据获取工具 🏆
🌐 网址: 开宝体育

## 📚 使用方法
1. 安装依赖 💻
```shell
pip install -r requirements.txt

#(可选)如果安装依赖太慢，可以使用镜像源哦~
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
2. 执行爬取操作 🚀
```shell
python main.py
```

<p>注意: 如果需要修改爬取的参数，请在 main() 函数中自行修改哦~ ✨<p>
